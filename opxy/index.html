OPVSCVLA → OP–XY a MIDI fork · non-equal temperament out to teenage engineering's OP–XY

The main collection synthesizes its music in the browser. This fork instead plays the OP–XY over MIDI, taking care to preserve just intonation and other non-equal tunings. Two proof-of-concept machines demonstrate the two ways to get a non-12-TET scale onto the device.

path A · tuning table

FADÓ

Pythagorean (3-limit) just intonation. Generates twelve detune values for the OP–XY's built-in tuning table, then sends plain MIDI notes — full polyphony, no bending. Best for any ≤12-note, octave-repeating scale.
path B · per-note bend / MPE

TRITAVA

The Bohlen–Pierce scale — no octave, so no tuning table can hold it. Each note gets its own channel and is bent to its exact frequency. For >12-note, non-octave, or drifting tunings.

Both pages carry an internal audio monitor (they play the exact target frequencies), so you can hear the result with no hardware attached. Web MIDI needs Chrome or Edge — Safari has none. See README.md for the OP–XY setup steps and the one capability (voice-per-channel MPE input) still worth confirming on hardware.
